Instructions For Installing The Code In Your Web Site
  • Choose Your Code Format After entering the necessary AdvertiPage customising information and submitting the form we will display the code for two different methods that produce the Advertipage opn your website.
    1. One format we deliver to you is the directory in a complete page template, ready to install but without your graphics and menu of course. The areas are well marked, however, so you can easily insert your own.
    2. The second format we deliver to you is the directory in two sections of code, ready to install into the <head>. and <body> sections of one of your own page templates.
  • Installing Format One
    • Save A Blank Page To Your Web Directory Folder
      If you have not created the folder in your website to hold the AdvertiPage yet now is a good time to do so. It, and the template page you are about to save in it must both be named as you name them in the configuration form. The page you save must have a php extension.
    • Copy and Paste The Code We Deliver To That Page
      Our template produces a standard 3 column with a header and footer web page. You will be able to replace those sections by copying them from pages on your website and pasting that into the appropriate sections on our page, leaving the web directory code behind and in between your pastings.
    • Copy And Paste Your Menus, Pictures etc Into the Page
      By comparing the various text displayed on the AdvertiPage to what is in the code you can locate the various sections of the page
      Exception - the code before the closing </head> tag in your supplied page should not be altered except for CSS style information.
      Exeception 2 - nothing between the code starting with "$file="http://" to "echo $data" should be changed
  • Installing Format Two
    • You Need A "Template" Of Your Webpage
      If you don't already have a template of your webpages a real simple way to get one is to use one of your most basic web pages on your site and copy its source code. Open the one you want to use with your browser and click the "View" link in the browser nav bar and the "Source" or "Page Source" link depending on which browser you use.Copy the source code into a text editor . We will paste our two sections of code into that page and replace the pages current content with our new directory.
    • Save The Template Page To Your Web Directory Folder
      If you have not created the folder in your website to hold the AdvertiPage yet now is a good time to do so. It, and the template page you are about to save in it must both be named as you name them in the configuration form. The page you save must have a php extension.
    • Copy And Paste The First Section Of Code (i.e.The Dynamic Page Title, Description and Meta Tag Section)
      All of those items reside in between the <head> </head> tags toward the top of your template page We are going to replace everything on your template page from the opening <head> tag back to the very first line and character of code. We are going to leave the closing </head> tag though, and perhaps some of your own style information if you like. You cannot leave the <title>, <description>, nor <keyword> tags however. If you do, the dynamic page feature will not work for those items and all your pages will have the same ones. After configuring your script and submitting the form copy the first section from the code we deliver to you and paste it into your template page as described above, being sure to leave the closing </head> tag in place.
    • Copy And Paste The Second Section Of Code
      This sectioon of code produces the AdvertiPage Web Directory
      Locate the opening <body> tag and the closing </body> body tag in your template.
      You can paste this second section of code over any content in your template that is in between those two body tags.
  • Check For Errors
    Open the page in your browser. You should have the main category page displaying the categories
    If so, click on one of the categories to open up its sub-categories.
    You should have a page displaying the sub-categories and links that have been submitted to it previously
    If everything is working, congratulations! You now have a web directory for your web site visitors and potential income.
    If not, please consult the documentation or the forums. It usually is some simple problem and there are just a few simple fixes to get it working.
