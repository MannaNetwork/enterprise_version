What is BungeeBones?
  • At its core, BungeeBones is a website promotion and traffic generating system.
  • It is a Web 2.0 php web directory script developed by Robert Lefebure to add interactive website advertising and promotion to web sites through its "distributed" web directory script.
  • It is a community of webmasters and website owners that use the script and cooperate to promote each other's websites in a manner similar to a link exchange.
  • It is a website income generator through a system similar to, but still quite different from, Pay Per Click
  • It is a combination of tried and well proven web strategies such as link exchange, web directories, reciprocal linking, Search Engine Optimisation and Pay Per Click
The Difference Between BungeeBones & Other Directories?
  • BungeeBones is completely different from other web directories because it is "distributed" and gets installed (complete with links and categories) on many web sites. The BungeeBones AdvertiPage combines the concepts of Web 2.0, Pay Per Click (the most used web advertising format), link exchange (the time tested foundation of the web) and web directories into a completely new model of website advertising and promotion. As each new distributed web directory gets installed it acts as a "portal" into the network of advertising websites. We can say that its web traffic is now added to the pool of traffic from the other websites that an advertised link in the directory is exposed too. One new distributed web directory after another each add to what in print media is called the "circulation" numbers. As that number increases so too does the value (and the cost to the advertiser) of advertising in the network. The Difference Between BungeeBones & Pay Per Click?
  • BungeeBones is a "Pay Per Action" model

    We believe Pay Per Action is fairer to webmasters. When you install the distributed web directory you get paid every time someone takes the "action" of subscribing to a paid link position

  • BungeeBones Pays Through Multiple Levels

    If someone who joins BungeeBones from your site adds a distributed web directory to their site BungeeBones pays you overide commissions on their ad sales too. There will be only two types of visitors to your distributed web directory - those with their own website and those without one. Since BungeeBones's product is web traffic both of us are interested in getting the former group to join BungeeBones. We do offer various levels of membership (including free) but when someone adds a distributed web directory too they are helping us grow and you receive your share of that growth.

  • BungeeBones Pays Residual Income

    You earn commissions EACH time the advertiser advertises, not just one time.

  • More respectful Of Your Customers/Visitors

    BungeeBones does not place its links (or ads) on the same page as your content or products. Rather, the Distributed Directory is installed on its own page leaving your site visitor to review your site uninterupted and without other distractive ads from competitors.

Instructions For Adding Your Link
  • Steps To Add Your Link To BungeeBones
    1. Use the directory below to migrate to the category you want to add your link to.
    2. If you feel adding regional information to your link listing is a benefit use the four dropdowns that will appear (after you select from the main categories) to select your continent, country, state, city. Ignore them if regional info does not apply to your website situation.
    3. If your feel your category is not listed then select to the closest one we offer and add your own suggestion. We appreciate your ideas and will do everything we can to accomodate your request. If the suggestion is approved we are glad to move your link into the new category/region for you.
    4. If you are satisfied that you have arrived where you want to place your link then scroll down to the very bottom of the page to add your link and other information to the form, perhaps noticing the other links in the category as you scroll. we also rely on this to help BungeeBones achieve the top quality it is aiming for by enabling new registrants to notify us of poor links and sites and rating them according to their relevancy to the category. Not only does "Peer Rating" help improve the directory's quality but you also get to see other websites and what features they have.
    5. Lastly, the script will present a confirmation page before final submittal. At that time it may also provide additional information, if applicable, such as the number of paid/free links in the category and/or the regions. if you added information in the optional section to create a Distributed Web Directory for your website it will also deliver the code that you can then copy and paste into your web page template.
