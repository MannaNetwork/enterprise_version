Instructions For The Configuration Form (Below)
  • Steps To Configure Your Distributed Web Directory To Function On Your Website
    1. Create A New Folder On Your Website And Enter The Folder Name (aka a directory) In The Form Below
      Create a new folder in your website to hold your new web directory web page (i.e. www.Your-domain.com/your_new_BungeeBones_web-directory_folder/).

      A SEO tip: relevant (to your topic) folder names can boost search engine rankings.

    2. Create A New Page Where Your Web Directory Script Will Be Copied To And Enter The Name In The Form Below
      A basic page name like "index.php" is fine but you can name it anything. Note the php file extension. The file must have a php extension. The final address to your new page would be something like www.Your-domain.com/your_new_BungeeBones_web-directory_folder/your-page-name.php
    3. Create And Enter Your Dynamic Page Title
      BungeeBones lets you make every display page (as in every page for every different category) have a different title that emphasizes that category. This is a very important SEO feature. You need to enter two phrases in the form below to be used in your page's META TITLE (which also is displayed in the status bar at the top of each page). These two phrases will be coordinated and combined with each category name to produce unique titles for every web directory page. And since you have created them, they are also customized to your website. We suggest something like your site's domain name (if descriptive) and/or main topics or keywords are best, as search engines place a lot of weight on titles for their results placement. Our script will add the category name after your custom TITLE 1 and before your custom TITLE 2 sections so that each page's title for each category page is unique both to your site and unique for each particular category

    4. Add Your Meta Description
      Some search engines use your description in their search results while others will pull it from the content of the page itself. It should not be over 200 characters and we will add the category name to the end on every description to make it unique on every page.
    5. Do You Want To Display Free Links?
      As a widget directory host you get to choose whether or not to display the links of websites that are in our free program.

Instructions For Installing The Code In Your Web Site
    This is just a preview of the installation instructions. The complete instructions will be supplied when the form below is submitted and when the code is delivered to you.
  • You Will Have Three Sections Of Code To Paste Into A "Template" Of Your Web Page That You Supply
    1. You Need A "Template" Of Your Webpage
      If you don't already have a template for your webpages making one is not too difficult.
    2. Place The Template Page In Your New Web Directory Folder
      The template page you are about to paste the code in must be named as you name it in the configuration form below. The page you save must have a php extension.
    3. Copy And Paste The First Section Of Code In The "Head" Section Of The Template Page
      The first section of code is to be pasted in place of everything before the closing </head> tag (at the top of your template page). We are going to show you how to locate the closing </head> tag and explain how to perhaps add some of your own style information if you need that.
    4. Copy And Paste The Second Section Of Code In The "Body" Section Of The Template Page
      This section of code produces the BungeeBones Web Directory
      The second section of code is to be pasted in between the opening <body> tag and the closing </body> body tag in your template.
      You can paste this second section of code over any content in your template that is in between those two body tags.
    5. Copy And Paste The Third Section Of Code In A "SideBar" Of The Template Page
      This section of code produces the optional "reciprocal" or link exchange component between you and the other webmasters with directories installed. This code will display a more prominent link (in your sidebar) of other webmasters that also have the distributed web directory installed. They will be displayed only in their category's sidebar. Your site will, in turn, be displayed on their site's sidebar, in your site's category.
      The third section of code is to be pasted in between the opening <td> tag and the closing </td> tags that create the sidebar in your template. That could be either the left or right sidebars according to your preference.
    6. What Should Happen
      After installing the code, when you open the page that you installed (i.e. with your browser) you should have the main category page displaying.
      If so, the links of any one of the categories to open up its sub-categories should have as its URL your website address. The directory appears as if it is all part of your website.
      Clicking a category link should produce a page displaying the sub-categories any links that have been submitted to it. The links and categories will be maintained and updated for you free of charge by Bungeebones and you will have zero maintenance to perform in maintaining the directory
